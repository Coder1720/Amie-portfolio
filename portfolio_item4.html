Hurricane analysis

Organising and Manipulating Data

While doing this project I experimented, practiced and learned alot more about parameters, conditionals, lists, dictionaries, string manipulation, return statements and how to use them.

In this project a default list of hurricanes were converted to dictionaries organised by Name, year,etc using keys. Then the list and dictionaries where iterated through and different conditionals were set to calculate different values such as how many times a specifc area was afected by a hurricane, hurricanes that caused the most damages, hurricane ratings based of mortality scale etc.
